## RU Описание

ЧёПоесть - электронная кулинарная книга, которая позволяет хранить и редактировать рецепты в простой, понятной форме.
Он включает в себя:
- Простой интерфейс для работы с рецептами
- Полный функционал для редактирования и поиска рецептов
- Хранение рецептов в базе данных, используя SQL-таблицу
- Возможность работы с несколькими базами данных

Использованные технологии:
- Python 3.x
- Библиотеки:
  - PyQt6
  - SQLite3
  - Random

## RU Ожидания от проекта
- Создания удобной электронной кулинарной книги
- Реализация функций: создания, редактирования, удаления и хранения рецептов, генерации списка покупок
- Реализация перевода на другой язык

## RU Начало работы

### Установка

1) Откройте файл `WhatTheEat_Release.exe`.
2) Нажмите кнопку `View raw`.

Если Windows будет блокировать приложение, нажмите "Подробнее", затем "Выполнить в любом случае".

### Настройка программы

Для изменения языка на английский нажмите кнопку "RU" в верхнем правом углу, для изменения языка обратно нажмите кнопку "EN" в том же углу.

Для создания/открытия базы данных нажмите сверху кнопку "Файл", затем выберите "Новая база данных" для создания или "Открыть базу данных" для открытия базы данных.

Альтернативно можно нажать Ctrl + N для создания или Ctrl + O для открытия базы данных.

## EN Getting started

### Installation

1) Open file `WhatTheEat_Release.exe`.
2) Click `View raw`.

If Windows will block the app from launching, press "More info", then press "Run anyway".

### Setting up the program

To change the language to English, click the "RU" button in the upper-right corner, and to change the language back, click the "EN" button in the same corner.

To create/open a database, click the "File" button at the top, then select "New Database" to create or "Open Database" to open the database.

Alternatively, you can press Ctrl + N to create or Ctrl + O to open the database.
